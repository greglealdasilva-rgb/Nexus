Nexus

Crie assinaturas proteicas e analise afinidades em segundos utilizando IA (ESM2).

Sobre

O Nexus é uma ferramenta de bioinformática para predição de interações entre sequências de aminoácidos. Utilizando o modelo facebook/esm2_t6_8M_UR50D, ele gera representações vetoriais de proteínas para calcular afinidades com alta precisão.

  • Banco de dados local em JSON.
  • Exportação de relatórios em PDF.
  • Interface retrô otimizada para fluxo de trabalho.

Nota de Segurança: Este software é destinado a fins de pesquisa acadêmica e proteômica. Os receptores cadastrados não possuem ligações com patógenos virais.
