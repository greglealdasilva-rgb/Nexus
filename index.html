Nexus

Crie assinaturas proteicas e analise afinidades em segundos utilizando IA (ESM2).

Nexus 1.0

Necessario Python | Versão Estável | Windows 10/11

Sobre

O Nexus é uma ferramenta de bioinformática para predição de interações entre Proteinas por sequências de aminoácidos. Utilizando o modelo ESM-2, ele gera representações vetoriais de proteínas para calcular afinidades com alta precisão.

  • Não precisa saber programar.
  • Exportação de relatórios em PDF.
  • Interface Simples.

Nota de Segurança: Este software é destinado a fins de pesquisa acadêmica e proteômica. É opensoucer com licença MIT.
